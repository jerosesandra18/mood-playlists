# 🎵 Mood Playlists Portal

An aesthetic, responsive bento-box style web application that acts as an interactive launching pad for curated musical eras. Built with native HTML5, modern CSS architectures, and dynamic JavaScript layer engines.

🚀 [**Launch Live Web Application**](https://my-mood-playlist.netlify.app/)

---

## ✨ Features

* **Dual Era Selection:** Seamlessly pivot between distinct cultural aesthetics (**Y2K Desi Romance** and **2012 Hollywood Pop**).
* **Bento Box Layout:** A modern, clean, fluid grid design styled completely with reactive Tailwind CSS.
* **Smart Link Search Engine:** Features a failure-proof universal deep-linking protocol that queries music platforms directly to bypass local browser sandbox security blocks.
* **Hardware-Accelerated Fluid UI:** Smooth screen state toggles utilizing hardware-accelerated fluid keyframe animations.

---

## 🎨 Design & Aesthetic Concepts

### 🍁 Y2K Desi Romance
* **Visual Vibe:** Chiffon sarees, handwritten poetry, and intense eye contact in midnight rain.
* **Color Profile:** Rich amber hues blending into organic stone transitions.

### 📸 2012 Hollywood Pop
* **Visual Vibe:** Varsity jackets, polaroid flash photography, and neon Tumblr layouts.
* **Color Profile:** High-energy pink accents contrasting against deep galactic indigos.
